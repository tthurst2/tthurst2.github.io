Quantum Gauntlet is 3D isometric local multiplayer arena brawler. It supports up to four players playing in Free For All or teams.

This was built in ~2 months Unity for GA Tech's CS6457. The game consists of a custom-built physics engine and combat state machine.

No pre-used assets; All models, particle effects, shaders, sound effects, and music done from scratch! The game only has one character, but it otherwise mostly feature complete featuring 4 player local multiplayer with CPUs as FFA or Teams with customizable stocks and two levels.

You can download it and try to run it via github, or I have hosted a webGL build you can load in browser. (Controller support works in browser but seems to be buggy on Firefox)
